Rifold E-Commerce

Full-Stack Development • Digital Transformation • Strategic Academic Project
Launch Live Demo

Role

Full-Stack Developer (Volunteer)

Timeline

Aug — Nov 2025

Tech Stack

Laravel, Tailwind CSS, MySQL, VPS

The Challenge

Identifying a local clothing brand with zero digital presence and initiating its full digital transformation as a strategic academic project.

Rifold previously relied entirely on manual sales or third-party marketplaces. They needed a dedicated identity and a platform that could handle a product catalog, shopping cart, and order management end-to-end to reduce operational steps.

The Solution

I delivered a comprehensive e-commerce solution, bridging the gap between high-fidelity UI design in Figma and a robust Laravel backend.

The platform was deployed on a custom VPS, ensuring full control over the server environment. This project was completed within a strict 3-month timeline while I maintained peak academic performance (GPA 3.84).

Key Deliverables

Full Transformation

Transitioned store operations from 100% manual to digital end-to-end.

VPS Deployment

Configured and deployed to production on a custom VPS for professional-grade hosting.

Core E-Commerce

Features include dynamic product catalog, cart system, and admin order management.

Responsive Design

Built with Tailwind CSS for a seamless experience across all screen sizes.

Technical Insight

Using Laravel allowed for rapid development of the order management logic, while Tailwind CSS ensured the "Modern & Clean" brand identity of Rifold was accurately translated from the Figma prototype to the browser.
